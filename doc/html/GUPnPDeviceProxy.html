GUPnPDeviceProxy — Proxy class for remote devices.

Synopsis

Object Hierarchy

  GObject
   +----GUPnPDeviceInfo

Description

GUPnPDeviceProxy allows for retrieving proxies for a device's subdevices and services. GUPnPDeviceProxy implements the GUPnPDeviceInfo interface.
